## Course Description

This course covers the study of intelligent agent design and rational decision making. Topics include goal-driven agents, search techniques, optimization, basic problem-solving methods, logic, knowledge-based agents, statistical and probabilistic reasoning, and the basics of machine learning. Upon completion, students should be able to demonstrate artificial intelligence design concepts.

**Prerequisite:** CSC-113

---

## Course Outcomes

Upon completion of this course, students will be able to:

- Create a custom agent
- Demonstrate foundational knowledge of deep learning
- Apply AI frameworks including Keras and PyTorch
- Implement commonly used AI algorithms such as classification and regression
- Apply the machine learning workflow to train a small model
- Apply computer vision fundamentals
- Implement natural language processing algorithms
- Use and optimize large language models
